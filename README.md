generator-ms-docker
==============================

This is a mircro service generator for docker.

## Support language

* [python (flask)](docs/flask.md)

## docker

* [peter1209/pyenv-uwsgi-run (python)](https://hub.docker.com/r/peter1209/pyenv-uwsgi-run/)

## Install

```bash
npm install -g generator-ms-docker
```

## Run

```bash
yo ms-docker
```
